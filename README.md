# Order Pizza Online

##### Takes Order information and prints order details and total.  _09/18/15_

#### By _**Rick Hills**_

## Description

_Order Pizza Online takes user inputs for Name, Size, Quantity, and Topping options, and creates your order and prints it to the page with the total cost._

## Setup

* Download or clone this repository from GitHub
https://github.com/r-hills/pizza.git
* Open index.html in a browser.  

## Technologies Used

JavaScript, JQuery, Mocha, Chia, HTML & Bootstrap
